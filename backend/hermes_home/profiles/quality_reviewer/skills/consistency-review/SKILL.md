# 一致性审查

审查权利要求与说明书的一致性。

## 审查要点

1. **术语一致性** - 权利要求和说明书使用相同术语
2. **特征对应性** - 权利要求特征在说明书中有对应描述
3. **范围一致性** - 保护范围与说明书公开内容一致
4. **引用一致性** - 附图标号、公式编号一致

## 常见问题

- 权利要求使用的术语在说明书中未出现
- 说明书描述与权利要求限定不一致
- 附图标号错误或不对应
