# 形式合规审查

检查专利申请文件的格式和形式合规性。

## 审查要点

1. **格式规范** - 字体、字号、页边距、行距
2. **编号规范** - 权利要求编号、段落编号
3. **术语统一** - 技术术语前后一致
4. **引用关系** - 从属权利要求引用正确

## 常见问题

- 权利要求编号不连续
- "所述"使用不规范
- 术语前后不一致
- 附图标号错误

## 工具使用

- `compliance_checker` - 检查文件形式合规性
