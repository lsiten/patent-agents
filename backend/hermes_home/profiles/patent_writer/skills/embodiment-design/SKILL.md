# 实施例设计

设计能够充分支持权利要求的实施例。

## 设计原则

1. 实施例要覆盖独立权利要求的所有技术特征
2. 提供多个实施例展示不同应用场景
3. 具体参数和数值要合理可实现
4. 实施例之间要有差异化

## 实施例层次

- **基础实施例** - 最简方案，覆盖核心特征
- **优选实施例** - 包含从属权利要求的特征
- **扩展实施例** - 展示不同应用场景
