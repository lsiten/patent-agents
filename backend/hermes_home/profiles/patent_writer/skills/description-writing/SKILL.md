# 说明书撰写

撰写公开充分、完整的专利说明书。

## 说明书结构

1. **技术领域** - 明确发明所属技术领域
2. **背景技术** - 描述现有技术及其不足
3. **发明内容** - 技术问题、技术方案、有益效果
4. **附图说明** - 简要说明附图内容
5. **具体实施方式** - 详细描述实施例

## 撰写原则

- 说明书必须公开充分，使本领域技术人员能够实现
- 实施例要充分，覆盖权利要求的所有技术特征
- 技术术语要前后统一，定义清楚
- 结合检索分析报告，突出创新点，规避现有技术

## 工具使用

- `description_writer` - 辅助生成说明书各部分内容
  - section_type: technical_field / background / summary / drawings / detailed
