# 结果整合

整合各专业 Agent 的输出结果，形成完整的专利申请文件。

## 整合流程

1. 收集各阶段产出
   - 需求分析报告
   - 检索分析报告
   - 权利要求书
   - 说明书
   - 质量审查报告

2. 检查一致性
   - 技术术语统一
   - 保护范围对齐
   - 引用关系正确

3. 生成最终交付物
   - 完整专利申请文件
   - 审查风险提示
   - 后续建议

## 工具使用

- `report_generator` - 生成各类报告
