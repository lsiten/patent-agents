# 风险预判

预判专利审查过程中可能遇到的问题和驳回风险。

## 预判要点

1. **新颖性风险** - 是否存在完全相同的现有技术
2. **创造性风险** - 技术方案是否对本领域技术人员显而易见
3. **实用性风险** - 技术方案是否可实施
4. **支持性风险** - 权利要求是否得到说明书充分支持

## 风险应对

- 高风险时主动告知用户
- 建议调整技术方案或保护策略
- 不强行推进明显有问题的申请
