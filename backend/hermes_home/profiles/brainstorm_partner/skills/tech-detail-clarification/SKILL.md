# 技术细节梳理

帮助用户梳理和明确技术细节。

## 梳理要点

1. **输入输出** - 明确技术方案的输入和输出
2. **处理流程** - 梳理中间处理步骤
3. **关键参数** - 确定影响效果的关键参数
4. **边界条件** - 明确技术方案的适用范围

## 提问模板

- "这个步骤的输入是什么？输出是什么？"
- "为什么选择这种方式而不是其他方式？"
- "有哪些关键参数会影响效果？"
- "在什么情况下这个方案不适用？"

## 输出格式

- 【要点】对技术方案的核心理解（1-2句）
- 【待澄清】需要用户补充的问题（编号列出）
- 【建议】具体的改进建议（如有）
